💡 VERWENDUNGS-BEISPIEL:
───────────────────────────────────────────────────────────────

Szenario: Du findest eine API-Dokumentation

1. Du bist auf: https://stripe.com/docs/api
2. Plugin öffnen → "Aktuellen Link speichern"
3. Titel: "Stripe API Docs"
4. Beschreibung: "API-Key: pk_test_123abc
                   Endpoint: /v1/charges"
5. Speichern ✅

Später brauchst du den API-Key:

1. Plugin öffnen
2. Auf "Stripe API Docs" klicken
3. 🎉 Seite öffnet + API-Key in Zwischenablage
4. Cmd+V → Sofort einfügen!
